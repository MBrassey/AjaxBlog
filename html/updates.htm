Roboss, Fun With Roblox Automation

I wrote an application to keep Roblox characters awake and earning coin in any minigame, it turned out to be especially useful for Meep City.

I woke up this morning and went to my office to find my character in Meep City awake with my wife's character seated next to me on the swingset in my backyard. They had been online all night and earned several thousand Meep Coin. Obviously this is not usually possibe. Typically one is logged out of a Roblox game after 20 minites of inactivity, or in this case would have fallen asleep after 5 minites in Meep City. If you leave it on for a day and you have a PLUS membership you recieve something like 20,000 Meep Coin. Considering this number of coin costs roughly $3,000 Robux or $35 in value, I started to feel like it was worth all the time coding Roboss.

Roboss is a stand alone application that sends a signal to your character in Roblox to ensure they never sleep. In my original design, I was experimenting with having my character speak wise sayings automatically so I kept this feature in the final release. Yes, it can make your character dance in any minigame as well. I implimented randomized delays between awake and speach signals to avoid detection and simulate an organic human being like behaviour, in this way Roboss is completely undetectable.

If you want to buy castles and fill them with every item in the game for epic parties, without spending your entire life and savings playing a kids game, you will need a source of Meep Coin, Roboss is definately an effective way to do it.

I released Roboss opensource with the option to buy the pre-compiled executible for $5 on my GitHub.



What it does:

function awake() {

... Sends signal to Roblox ensuring your character does not sleep ...
... Avoids detection algorithms by randomizing the delay between awake signals ...


}

function sayings() {

... Engages the Roblox window and automatically types a wise message from time to time ...
... Avoids detection algorithms by randomizing the delay between sayings ...
... Respectfully stops speaking after 200 cycles and re-activates the speaking button ...


}

function dance() {

... Makes Roblox character dance, if dancing is interrupted, it will begin again at a randomized interval ...
... Continues dancing until the StopFunc() is evoked via the "Stop" button ...


}

What Roblox games does it work on?

Roboss works on all Roblox games. It works especially well for games that require your active participation to stay awake.

How long will my character stay awake?

Roboss will keep your character awake as long as possible. Roboss also prevents network disconnection due to inactivity. If the Roblox server is updated or you have a physical or ISP related network disconnection, Roboss will stop working.

How much Meep Coin can I earn with Roboss?

Based on recent testing, Roboss deliveres around 20,000 Meep Coin per day to PLUS Members.

Will my account get banned?

No. Roboss uses methods to produce organic like activity simulating a real human being. It is completely undetectable.

Is Roboss Free?

Roboss is 100% Free & Open Source. If you find Roboss useful please clicking the donate button on my GitHub page ^__^.

Why did I make Roboss?

I made Roboss for fun, specifically for use in MeepCity.

How many Robux can you save by using Roboss?

In MeepCity, Roboss earns you $3,000 Robux worth of MeepCoin, this is $35 value per day!

Troubleshooting:

If you are already dancing, you have to select the dance option to continue staying awake (in some minigames). Roboss will continually activate the Roblox window, so you will not be able to have a window on top of Roblox. You can use a second monitor with Roboss and Roblox open so you can continue using your computer normally on the other. When using the "Speak Wisely" function, if you click on another window at the same time as Roboss is trying to type the message in to Roblox, the text will instead be printed on the window you clicked.
